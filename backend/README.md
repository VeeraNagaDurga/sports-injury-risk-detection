# Sports Injury Risk Detection - Backend

## Overview

The backend provides REST APIs for authentication, athlete profile management, video upload, pose estimation, and injury analysis.

## Features

- FastAPI
- MediaPipe Pose Detection
- Video Processing
- Athlete Profile APIs
- Authentication APIs

## Technologies

- Python
- FastAPI
- OpenCV
- MediaPipe
- Pydantic

## Run

pip install -r requirements.txt

python -m uvicorn main:app --reload

Runs on:
http://127.0.0.1:8000
