# Sports Injury Risk Detection - Frontend

## Overview
The frontend provides an interactive dashboard for athletes, coaches, physiotherapists, sports scientists, and administrators.

## Features
- User Authentication
- Athlete Profile Management
- Video Upload
- Dashboard
- Injury Analysis Results
- Responsive UI

## Technologies
- React.js
- React Router
- Axios
- React Icons
- CSS3

## Run

npm install
npm start

Runs on:
http://localhost:3000
